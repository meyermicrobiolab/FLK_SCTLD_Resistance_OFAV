# FLK_SCTLD_Resistance_OFAV
Scripts and R code to reproduce bioinformatics and data analysis workflows for the investigation of microbial contributions to stony coral tissue loss disease resistance in *Orbicella faveolata* corals

## Sections
 * **bioinformatics_scripts** - This is a section containing scripts used for generating the coassembly and Metagenome Assembled Genomes, so that all parameters for programs are accessible.      
 * **analysis_code** - This is a section with markdown documents for each main figure and analysis, so the manuscript analyses are reproducible    
 * **data** - This section contains data tables produced and used by the analysis code     
 * **figures** - This section contains figures produced by the analysis code
