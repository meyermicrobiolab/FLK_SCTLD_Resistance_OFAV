# Markers of resilience to stony coral tissue loss disease and probiotic potential in the microbiome of the threatened coral, _Orbicella faveolata_

Cynthia C. Becker, Allison R. Cauvin, Karen L. Neely, Caroline Dennison, Andrew C. Baker, Brian K. Walker, Julie L. Meyer

Scripts and R code to reproduce bioinformatics and data analysis workflows for the investigation of microbial contributions to stony coral tissue loss disease resistance in *Orbicella faveolata* corals

## Sections
 * **bioinformatics_scripts** - This is a section containing scripts used for generating the coassembly and Metagenome Assembled Genomes, so that all parameters for programs are accessible.      
 * **analysis_code** - This is a section with markdown documents for each main figure and analysis, so the manuscript analyses are reproducible    
 * **data** - This section contains data tables produced and used by the analysis code     
 * **figures** - This section contains figures produced by the analysis code 

## Instructions to Reproduce Figures & Analysis
* Download data files from Zenodo: https://doi.org/10.5281/zenodo.17947166   
* Figure 3 - Beta diversity analysis. Code additionally covers content in Figure S2, Figure S4 and Table S4.    
* Figure 4 - Differential abundance analysis. Code additionally covers content in Table S5.    
* Figure 5 - Antibiotic resistance gene analysis. Code additionally covers content in Table S6 and Figure S6.      
* Figure 6 - Biosynthetic gene cluster analysis. Code additionally covers Figure S5
